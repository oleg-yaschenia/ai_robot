# 00 Архитектура и стандарты (КАНОН)

## Цель
Единые интерфейсы, питание, разъёмы, чтобы интеграция не ломалась.

## Роли вычислителей
- Jetson: AI/CV/SLAM/высокоуровневая логика
- ESP32: моторы/энкодеры/датчики/безопасность/сторожевой таймер

## Канон шин
- UART Jetson<->ESP32: (скорость/формат) см. docs/05_UART_протокол.md
- I2C: (кто мастер, через что разведение) см. docs/04_I2C_адреса_и_шины.md
- USB: список устройств и питание

## Канон питания
Рейки: VBAT_3S / 12V / 5V_MAIN / 5V_SENS / 6V_SERVO / 3V3_LOGIC
См. docs/02_Карта_шин_и_питания.md

## Канон именования
Разъёмы: J1..Jn, жгуты: C-001.., линии: PWR_*, UART_*, I2C_*

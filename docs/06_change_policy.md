# 06 Правила изменений (чтобы не ломать интеграцию)

## Что считается "ломающим" изменением
- смена pinout / разъёма / уровня питания
- смена адреса I2C / канала TCA
- изменение формата сообщений UART без MAJOR-версии

## Процесс
- Любые изменения канона: PR + запись в CHANGELOG.md
- Версионирование протокола: SemVer (MAJOR.MINOR.PATCH)
